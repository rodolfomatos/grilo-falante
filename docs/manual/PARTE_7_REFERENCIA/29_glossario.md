# 29. Glossário

## Termos常用

| Termo | Definição |
|-------|-----------|
| **Claim** | Uma afirmação que pode ser verificada |
| **GMIF** | Governance Markup Interpretation Framework - sistema de classificação M1-M7 |
| **GF-ID** | ID único para claims (formato: GF-YYMMDD-TYPE-HASH) |
| **Gap** | Lacuna de conhecimento - algo que o sistema não sabe |
| **PINA** | Protocol for Normative Incorporation |
| **NCA** | Normative Candidate - regra extraída de conteúdo |
| **INVARIANT** | Regra que governa o raciocínio após incorporação PINA |

---

## Estados

| Estado | Significado |
|--------|-------------|
| **INACTIVE** | Sistema parado |
| **LOADED** | Regime carregado |
| **GOVERNING** | Regime ativo e a governar |
| **HIBERNATED** | Regime suspenso |

---

## GMIF Levels

| Level | Nome | Significado |
|-------|------|-------------|
| M1 | Primário | Múltiplas fontes confirmam |
| M2 | Contextual | Verdadeiro sob suposições |
| M3 | Parcial | Estrutura incompleta |
| M4 | Duvidoso | Contradições detetadas |
| M5 | Interpretação | Uma fonte clara |
| M6 | Derivado | Inferência lógica |
| M7 | Síntese | Agregado de fontes |
