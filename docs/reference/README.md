## Key Concepts Summary

### Lake / Island / Stone Metaphor
- **Lake** (Lago): The background field of memory and available information
- **Stone** (Pedra): A salient interaction that creates significance in the lake
- **Island** (Ilha): A consolidated cognitive aggregate - a "habitable semantic territory"

### Sleep/Wake Cycle
- **Sleep** (Ir Dormir): Batch processing - sanitization, aggregation around gravity centers, indexing
- **Wake** (Acordar): Restore sanitized context from islands, not just load vectors

### Core Thesis
> "Models don't learn; they can only be made learners by external governance."
> "The model executes; the regime learns."

### Key Distinction
> Correct factual answer ≠ Epistemic legitimacy
> Fluency ≠ Justification
