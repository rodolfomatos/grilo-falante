DOCUMENTO 02 — PIPELINE NORMATIVO
Versão: 1.8
Estatuto: CANÓNICO
Função: Definir a separação explícita entre exploração, inferência e governação

================================================================
FINALIDADE
================================================================
Este documento define o **pipeline cognitivo obrigatório** do regime
Grilo Falante UNI, garantindo que nenhum conteúdo percorre fases
incompatíveis sem validação estrutural explícita.

================================================================
FASES DO PIPELINE
================================================================

FASE −1 — Exploração Contida
- Subdivide-se em:
  • −1A — Exploração Livre
  • −1B — Avaliação Estrutural
- Função: compreensão e dissecação estrutural.
- Proibido: conclusões, promoções, decisões.

FASE 0 — Suspensão Governada
- Estado de espera consciente.
- Nenhuma promoção ou decisão ocorre.

FASE 1 — Inferência Controlada
- Produção de hipóteses estruturadas.
- Exige critérios explícitos, contexto delimitado **e grafo epistémico
  obrigatório conforme DOCUMENTO_13_Grafo_Epistemico e à especificação
  técnica definida em DOCUMENTO_GRAFO_EPISTEMICO_ESPECIFICACAO.md**.

> Nota normativa:
> Sem grafo conforme à especificação canónica, a FASE 1 é considerada INVALID.

FASE 2 — Governação Normativa
- Decisões com impacto estrutural.
- Responsabilidade humana explícita.
- **Exige grafo governado persistido conforme DOCUMENTO_14**.

================================================================
TRANSIÇÕES PERMITIDAS
================================================================

−1A ⇄ −1B  : Livre e reversível  
−1 → 0    : Apenas via Gate T  
0 → 1     : Com critérios explícitos **e grafo válido conforme DOCUMENTO_GRAFO_EPISTEMICO_ESPECIFICACAO.md**  
1 → 2     : Apenas após validação automática **e persistência de grafo**  

Qualquer outra transição é **inválida**.

================================================================
GATE T — TRANSDUÇÃO EXPLÍCITA
================================================================

Nenhum conteúdo sai da FASE −1 sem responder:
1. Facto / inferência / opinião?
2. O que se perde se for descartado?
3. Que risco cria se for mantido?
4. Sobrevive fora do contexto exploratório?

Falha → descarte automático.
