# Acoplamento Lint × AFS

## Princípio
AFS governa contexto. Lint governa inferência. Interposição obrigatória.

## Pontos
P0 Constructor: escopo e pressupostos
P1 Manifesto: exclusões justificadas
P2 Updater: inferências condicionais
P3 Output: segmentação + lint
P4 Evaluator: lint antes de escrever
P5 Persistência: apenas ACCEPT/CONDITIONAL

## Regra de inversão
AFS gere existência; lint decide validade.
