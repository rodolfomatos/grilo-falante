================================================================
FINALIDADE
================================================================
Este documento destina-se a utilizadores humanos que entram
no regime Grilo Falante UNI pela primeira vez ou após interrupção.

O objetivo é **usar corretamente**, não compreender tudo.

================================================================
PRÉ-REQUISITOS
================================================================
- Ler este guia do início ao fim.
- Não assumir continuidade sem ACORDAR.
- Não usar outputs exploratórios como decisões.

================================================================
FLUXO MÍNIMO DE USO
================================================================

1. **Novo projeto ou novo chat**
2. Escrever explicitamente: `ACORDAR`
3. Confirmar data e hora
4. Ler estado do BACKLOG
5. Declarar intenção do ciclo

Só após estes passos é permitido:
- explorar;
- inferir;
- decidir (com as regras apropriadas).

ERROS COMUNS

- “Assume que…”
- “Já sabes o que quero dizer”
