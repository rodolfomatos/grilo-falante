Versão: 1.7
Estatuto: OPERACIONAL
Função: Registo de cápsulas conceptuais estáveis e reutilizáveis

================================================================
O QUE É UMA CÁPSULA
================================================================
Uma cápsula é uma unidade conceptual mínima que:
- resulta de exploração ou auditoria;
- foi explicitamente validada;
- é suficientemente estável para reutilização futura;
- não depende de contexto conversacional específico.

Nenhuma cápsula é criada automaticamente.
