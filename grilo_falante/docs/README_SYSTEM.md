README_SYSTEM — system.tgz
Versão: 1.7.2
Estatuto: META (operacional, não normativo)

================================================================
FINALIDADE
================================================================
Este ficheiro identifica e contextualiza o pacote `system.tgz` do
regime Grilo Falante UNI.

O pacote serve exclusivamente para **transporte e reconstrução
material** do sistema no canvas, quando invocado pelo INSTALLER (A00).

================================================================
CONTEÚDO DO PACOTE
================================================================
O `system.tgz` contém apenas:
- documentos canónicos A01–A12;
- metadocumentos necessários à integridade
  (ADN.md, MANIFEST.lock, CHECKLIST_DE_INTEGRIDADE.md, CHANGELOG.md);
- este README.

Nenhum documento sombra, output de chat ou conteúdo exploratório
é incluído.

================================================================
MODO DE USO
================================================================
- NÃO abrir nem editar manualmente os ficheiros do pacote.
- NÃO usar fora do INSTALLER.
- NÃO tratar o pacote como fonte de verdade cognitiva.

Uso correto:
1. Colocar `system.tgz` no canvas do projeto.
2. Invocar `ACORDAR`.
3. Permitir que o INSTALLER execute o BOOTSTRAP, se necessário.

================================================================
AVISO DE INTEGRIDADE
================================================================
Este pacote:
- não valida conteúdo;
- não cria autoridade;
- não substitui os documentos canónicos após materialização.

A integridade do sistema depende sempre do MANIFEST.lock e do
Documento 12 — Validação Automática.
