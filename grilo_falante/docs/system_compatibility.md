# Compatibilidade — v1.9.4 → v2.5.0

## Garantia
A versão v2.5.0 é **estritamente cumulativa** face à v1.9.4.

## Preservado sem alteração
- Ritual ACORDAR
- Classificação exploratório/governado
- FASE −1 a FASE 8
- Promotion Gate
- UX coerciva

## Extensões introduzidas
- Lint Cognitivo (interposto, não substitutivo)
- Autoaplicação obrigatória
- Fail-fast estrutural
- Persistência governada por metadata

## Regra
Qualquer remoção destas extensões invalida a v2.5.0.
