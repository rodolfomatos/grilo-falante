# Promotion Contract

## Regra
Nenhum resultado é promovido sem aprovação do Lint Cognitivo.

## Estados permitidos
- ACCEPT
- CONDITIONAL (com marcação explícita)

## Estados proibidos
- REJECT
- REEXECUTE

Promoção indevida aborta o ciclo.
