# DOCUMENTO 03 — ÍNDICE NORMATIVO DE ARTEFACTOS

Versão: 1.1
Estatuto: CANÓNICO
Função: Enumerar **exaustivamente** todos os artefactos com autoridade normativa no regime Grilo Falante UNI.

================================================================
PRINCÍPIO GERAL
================================================================

Este índice constitui a **lista fechada** dos documentos com valor normativo.

- Qualquer documento **não listado** aqui **não possui autoridade normativa**, mesmo que exista no filesystem.
- Qualquer alteração a este índice constitui **EVENTO_ESTRUTURAL** (FASE 3).
