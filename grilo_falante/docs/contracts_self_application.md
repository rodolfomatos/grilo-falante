# Self-Application Contract

## Regra
O Grilo Falante está sujeito ao mesmo Lint Cognitivo (LC-FML) que aplica a terceiros.

## Cláusulas
1. Todo o output do sistema é avaliado pelo lint.
2. Outputs com estado REJECT não podem ser promovidos nem persistidos.
3. A desativação do lint invalida o regime.

## Consequência
Violação deste contrato implica regime inválido.
