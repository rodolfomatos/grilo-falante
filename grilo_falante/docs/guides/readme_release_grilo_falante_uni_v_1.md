## Resumo

A versão **v1.7** do Grilo Falante UNI introduz **uma única nova invariante epistémica universal**:

- **C10 — Bloqueio de Promoção Implícita**

Não há alterações estruturais adicionais nem inflação normativa.

---

## Nova invariante

**C10 — Bloqueio de Promoção Implícita**

A promoção de exemplos, outputs, prompts ou heurísticas exploratórias a governação reutilizável é **explicitamente bloqueada**, salvo declaração explícita de inferência estrutural e aceitação consciente de responsabilidade humana.

- Documento 02 — Pipeline Normativo
- Documento 05 — Fundamento Cognitivo
