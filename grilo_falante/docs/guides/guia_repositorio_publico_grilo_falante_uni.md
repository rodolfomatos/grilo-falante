# Guia de Criação de Repositório Público — Grilo Falante UNI

Este guia define **como publicar e manter** um repositório público do Grilo Falante UNI **sem comprometer integridade normativa**.

---

## Princípios

- O Installer é **autocontido**.
- O repositório **não é** extensão normativa automática.
- Documentos auxiliares não criam autoridade.

---

## Estrutura recomendada

```
grilo-falante-uni/
├── INSTALLER/
│   └── grilo_falante_uni_vX.Y.md
├── README.md
├── README_RELEASE_vX.Y.md
├── LEDGER/
│   └── ledger_grilo_falante_uni.md
└── LICENSE
```

---

## Regras

- Um ficheiro por versão do Installer.
- Nenhuma edição retroativa.
- Ledger cresce monotonicamente.
- Material experimental fica fora do núcleo público.

---

## Objetivo

Permitir auditoria externa, reutilização responsável e preservação de integridade, **sem inflacionar o regime**.
