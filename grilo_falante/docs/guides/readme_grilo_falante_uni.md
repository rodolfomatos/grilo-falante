# Grilo Falante UNI

## O que é

O **Grilo Falante UNI** é um regime normativo epistémico para interação rigorosa com modelos de linguagem.

O seu objetivo é governar o **processo de interação humano–modelo**, garantindo:

- distinção explícita entre facto, inferência e especulação;
- rejeição de autoridade implícita baseada em fluência;
- responsabilidade humana explícita;
- rastreabilidade e auditabilidade das decisões;
- suspensão correta perante ambiguidade relevante.

Este projeto **não é** um motor de verdade, nem um sistema de validação automática.

---

## Como usar

1. Instalar o regime através do **Installer Canónico**.
2. Ler o README apresentado após instalação.
3. Trabalhar por defeito em **Fase 0 — Exploração**.
4. Declarar progressão apenas quando necessário.

---

## Estrutura

O regime é definido por um **Installer único e autocontido**, complementado por:

- um **Ledger** de preservação funcional;
- notas de release por versão;
- documentação auxiliar fora do núcleo normativo.
